Console Projects & Data Structures

Console Labyrinth (C++, 3rd Semester)

A grid-based ASCII labyrinth game developed in C++ during my 3rd semester. The player moves through a text-based map and tries to reach the exit.

The focus of this project was on input handling, grid-based logic, and maintaining game state within a console environment.

AVL Tree Implementation (C#, 2nd Semester)

A data structures project implemented in C# during my 2nd semester.

The application demonstrates insertion, deletion, searching and automatic rebalancing of an AVL tree, with a console-based visualization.
